# ✈️ SkyVoyage

**SkyVoyage** is an interactive travel website designed to enhance the user experience for European travelers by providing real-time weather forecasts for major cities. This project leverages HTML, CSS, JavaScript, and a reliable weather API to deliver accurate, up-to-date information in a user-friendly format.

---

## 🌟 **Project Scenario**

A European travel agency sought to boost bookings by providing visitors with a 7-day weather forecast for major European cities. The task was to create a webpage that retrieves weather information from an external API, processes the data with JavaScript, and displays the forecast using HTML and CSS.
